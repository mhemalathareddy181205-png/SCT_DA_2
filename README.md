# SCT_DA_2 - Data Cleaning and Preprocessing

## Project Overview

This project focuses on data cleaning and preprocessing using Python and Pandas. The objective was to improve data quality and prepare the dataset for further analysis by handling missing values, removing duplicate records, and ensuring data consistency.

## Objective

* Load and inspect the dataset.
* Identify and handle missing values.
* Detect and remove duplicate records.
* Verify data types and column structure.
* Clean and prepare the dataset for analysis.

## Tools and Technologies

* Python
* Pandas
* Jupyter Notebook

## Steps Performed

### 1. Data Loading

* Imported the dataset into a Pandas DataFrame.
* Explored the dataset structure and dimensions.

### 2. Data Inspection

* Examined column names and data types.
* Checked dataset information and summary statistics.

### 3. Missing Value Handling

* Identified null values in the dataset.
* Removed or handled missing records where necessary.

### 4. Duplicate Removal

* Checked for duplicate entries.
* Removed duplicate records to maintain data integrity.

### 5. Data Validation

* Verified the cleaned dataset.
* Confirmed that no unwanted missing values or duplicates remained.
